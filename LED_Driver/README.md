# STM32F407 LED Driver

A bare-metal LED driver developed for the STM32F407 Discovery board using Embedded C.

## Features

- LED initialization
- Turn LED ON
- Turn LED OFF
- Toggle LED state
- Register-level GPIO configuration

## Hardware

- STM32F407 Discovery Board
- On-board LEDs connected to GPIOD

## Driver Files

- `led.h` — Driver declarations and LED configuration
- `led.c` — Driver implementation
- `main.c` — Example application

## Status

LED driver implemented and tested successfully on hardware.
